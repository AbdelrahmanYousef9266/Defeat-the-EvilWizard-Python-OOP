## Overview

Defeat the Evil Wizard is a turn-based battle game built using Python and Object-Oriented Programming (OOP).

In this game, the player selects a character and battles against a powerful Evil Wizard. The goal is to strategically use attacks, special abilities, and healing to defeat the enemy before being defeated.

## Features
  Multiple character classes:
Warrior
Mage
Archer
Paladin
 Turn-based combat system
 Unique special abilities for each character
 Healing system with max health limit
 Randomized attack damage for dynamic gameplay
 Evil Wizard that regenerates health each turn
 Victory and defeat conditions
 Technologies Used
Python 3
Object-Oriented Programming (OOP)
Classes & Objects
Inheritance
Methods

## How to Play

Run the Python script:

python main.py
Choose your character:
Warrior (balanced)
Mage (high damage)
Archer (fast attacks)
Paladin (defense & healing)
During battle, choose actions:
Attack
Use Special Ability
Heal
View Stats
Defeat the Evil Wizard to win!
 OOP Concepts Used
Inheritance → Character → Warrior, Mage, Archer, Paladin
Encapsulation → Character attributes (health, attack power)
Polymorphism → Different behaviors for special abilities
